WordPress Information

To install this theme you must have a working version of WordPress already installed. If you need help installing WordPress, follow the instructions in WordPress Codex. Please ensure that you have the minimum requirements for the theme is WordPress 4.4.0 or highter, PHP 5.4 or highter and MySQL 5.4 or highter.

Recommended PHP Configuration Limits

Many issues that you may run into such as; white screen, demo content fails when importing, empty page content with Fusion Builder and other similar issues are all related to low PHP configuration limits. The solution is to increase the PHP limits. You can do this on your own, or contact your web host and ask them to increase those limits to a minimum as follows:

  • max_execution_time 180
  • memory_limit 128M
  • post_max_size 32M
  • upload_max_filesize 32M

Theme Installation

This instruction will spend for the customers who are the new one with WordPress – Kuteshop theme.

When downloading the package on Themeforest, they will have the theme package with version 3.x

WordPress Installation

Step 1: Go to Appearance > Themes > Add New Theme , then click Upload Theme button

Step 2: Go to Browser then select Kuteshop-v3.x.zip file

Step 3: Click Install Now to install Kuteshop

Step 4: Wait for a while until the installation complete then select Activate link to activate theme. Theme has been installed and activated. Beside that you can also click to Live Preview to view this theme

FTP Installation

Step 1: Log into your hosting space via an FTP software like File Zilla.

Step 2: Unzip the kuteshop.zip file and use only the extracted Kuteshop theme folder

Step 3: Upload the extracted kuteshop theme folder onto wp-content > themes folder.

Step 4: Activate the newly installed Nitro by going to Appearance > Themes, find kuteshop and click on “Activate”button

Guide to re-install theme version from 1.x to 3.x (For customers who are using theme version 1.x)

Please note that this part just spends for the customers who are using theme version 1.x and now would like to use version 3.x. Please read this section. If you are the new one with WordPress – Kuteshop theme, please read carefully Theme Installation part.

WordPress Update Installation

Notice: Your database and content will not be erased or lost during updating theme version 1.x to version 3.0.

Step 1: Deactivate your current Kuteshop theme version. Navigate to WordPress Dashboard > Themes. Then activate another theme.

Step 2: Delete the old Kuteshop theme. Hover your mouse over the Kuteshop theme thumbnail. Click “Theme Details”, then click the red “Delete” button.

Step 3: Get the Kuteshop.zip file you have downloaded from ThemeForest.
Note: if you have downloaded the main file, you have to unzip that file to get the Kuteshop-v3.x.zip theme file.

Step 4: Installing Kuteshop theme update. Navigate to WordPress Dashboard > Appearance > Themes.Click “Add New” button on theme list, click “Upload Theme”, then browse to Kuteshop-v3.x.zip file on your computer.

Step 5: Once the upload completes, click “Activate” to activate Kuteshop.

Note: If updating a theme, please do not use Import Simple Data. All your data will be lost or duplicate if you use this.

Negative to Dashboard > Pages > Add new

– Select Backend Editor

– Then click to Add Template button. It will allow you to add the default Template without building again all contents.

Then, you can add content of homepage you want to re-install (in Default Template tab). There, we have all available contents of all homepages. Please click on Plus icon to add

In case you want to delete the content, please change to Classic Mode

Select all the contents then Delete

FTP Update Installation

Notice: please backup all your theme files and database before re – installing

Step 1: Log into your server, then navigate to wp-content > themes folder. Then back up / delete old theme. The datas will be not erased or lost.

Step 2: Upload the Kuteshop-v3.x.zip file to wp-content > themes folder.

Step 3: Extract Kuteshop-v3.x.zip file into wp-content > themes folder.

Step 4: Go back to Dashboard > Appearance > Themes then Activate Kuteshop theme.
